## [1.0.0-rc.4] - 2026-06-18

**Initial open-source release.**

First public release of `@cisco_open/linting-document-store` under the Apache-2.0 license.

### Features

- Pluggable adapter pattern: Local, PassThrough, and MCP adapters.
- Upload history tracking with graceful degradation.
- Shared by the orchestrator and MCP analysis server.
